# PROJECT DESIGN SYSTEM

## Marca

**Nombre:** TE VES GUAUU  
**Rubro:** Peluqueria Canina

## Objetivo principal

Generar confianza y lograr que el visitante reserve un turno mediante WhatsApp.

## Personalidad de la marca

La identidad visual de TE VES GUAUU debe sentirse:

- Cercana
- Dulce
- Profesional
- Con amor por las mascotas
- Alegre
- Moderna
- Calida

## Sombras

Las sombras deben ser muy suaves, con preferencia por tonos rosados de baja opacidad.

Ejemplo conceptual:

- Sombra ligera para cards.
- Sombra sutil en hover.
- Evitar sombras duras, oscuras o demasiado marcadas.

## Espaciados

Escala base de espaciado:

- `8px`
- `16px`
- `24px`
- `32px`
- `48px`
- `64px`
- `96px`

## Animaciones

Las animaciones deben ser suaves, calidas y discretas:

- Fade al aparecer.
- Cards elevandose levemente.
- Botones con escala suave.
- Imagenes con pequeno zoom.
- Navbar reduciendo altura al hacer scroll.
- Boton flotante de WhatsApp con animacion muy sutil.

## Elementos graficos

El lenguaje visual puede incluir:

- Huellas.
- Corazones.
- Lineas dibujadas a mano.
- Estetica amigable.
- Esquinas redondeadas.

## Principios de diseno

La web debe transmitir limpieza, confianza, carino y profesionalismo.

Cada decision visual debe ayudar a que la experiencia se sienta clara, amable y confiable. La estetica debe comunicar cuidado real por las mascotas, sin perder orden ni calidad profesional.
